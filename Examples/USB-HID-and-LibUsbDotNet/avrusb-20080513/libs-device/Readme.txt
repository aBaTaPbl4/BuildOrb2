Это файл Readme для папки libs-device. Она содержит куски кода, которые используются для firmware USB-устройств.


ЧТО НАХОДИТСЯ В ЭТОЙ ПАПКЕ?
===========================

osccal.c и osccal.h
  Этот модуль содержит функции, которые калибруют встроенный RC-генератор микроконтроллера AVR, основываясь на тактах фрейма USB. См. osccal.h для документации API.
