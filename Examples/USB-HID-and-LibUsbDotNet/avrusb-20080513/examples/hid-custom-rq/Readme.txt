Этот файл Readme для примера hid-custom-rq. Суть примера та же, что и в примере custom-class example за исключением того, что USB-устройство относится к классу USB HID class.


ЧТО ЗДЕСЬ ДЕМОНСТРИРУЕТСЯ?
==========================
Этот пример демонстрирует как пользовательские запросы могут быть отправлены устройствам HID. Этот механизм может использоваться для предотвращения появления диалога "driver CD" в Windows, и работает он под управлением все той же библиотеки libusb-win32. Пример можно также использовать для расширения функциональности класса USB, например путем установки параметров.

Пожалуйста примите во внимание, что вы должны установить filter-версию библиотеки libusb-win32 для реализации всех возможностей этого режима. Версия "device driver" может получить доступ к устройству только в том случае, если оно было зарегистрировано с помощью соответствующего файла *.inf. Версия "filter" имеет доступ ко всем устройствам.
