Установка драйвера фильтра (Filter Driver)
- в версии 0.1.7.10 драйвер фильтров устанавливается с помощью GIU-инсталлера, который делает установку и деинсталляцию более простой и защищенной.
- залогиньтесь с привилегиями администратора.
- загрузите самый свежий инсталлятор (libusb-win32-filter-bin-x.x.x.x.exe). 
- перед установкой закройте все приложения, которые используют устройства USB. 
- запустите инсталлятор и следуйте его указаниям. Не запускайте инсталлятор с устройства памяти USB! 
- по окончании процесса инсталляции обычно запустится тест- программа. Она проверит правильность установки и напечатает дескрипторы всех устройств USB, которые доступны через библиотеку. Эта тест-программа может быть запущена позже из системного меню запуска (Start Menu). 
- перезагрузка не нужна. 

Установка драйвера устройства (Device Driver)
- в версии 0.1.7.10 драйвер устройства поставлятся как отдельный пакет, который включает все необходимое для использования libusb-win32 для одиночных устройств и обычного драйвера устройства. Инсталляция filter driver болше не нужна! 
- залогиньтесь с привилегиями администратора.
- загрузите самый свежий двоичный пакет драйвера устройства (libusb-win32-device-bin-x.x.x.x.tar.gz). 
- распакуйте все во врЕменную папку.
- переименуйте и отредактируйте пример inf-файла 'libusb.inf' для соответствия Вашему устройству (или устройствам) - модифицируйте vendor и product ID, строки и т. п.). Создайте различные inf-файлы для инсталлирования различных различных типов устройств (устройств с разными ID). 
- отключите устройство (устройства) от системы. 
- откройте Windows Device Manager (Диспетчер Устройств) и удалите все некорректно установленные USB-устройства (которые имеют желтый восклицательный знак). 
- снова подсоедините Ваше USB-устройство (или устройства) в систему. 
- когда Windows запросит драйвер, выберите inf-файл(ы), созданные ранее. На Win2k и WinXP система предупредит о том, что драйвер не 'digitally signed'. Проигнорируйте это сообщение и продолжите инсталляцию. 
- откройте Windows Device Manager (Диспетчер Устройств) для проверки, что USB-устройство корректно установилось в системе. Запустите программу теста (testlibusb-win.exe) из папки 'C:\Program Files\LibUSB-Win32\bin'. Программа должна напечатать жескрипторы Ваших USB-устройств. 
- перезагрузка не нужна. 

Удаление драйвера фильтра (Filter Driver)
- для удаления filter driver откройте Control Panel, откройте папку 'Add or Remove Programs', выберите запись 'LibUsb-Win32-x.x.x.x' и удалите её. 
- перезагрузка не нужна. 

Удаление драйвера устройства
- device driver не может быть удален из системы. Windows не предоставляет для этого каких-либо инструментов. 

Обновление драйвера фильтра (Filter Driver)
- если у Вас установлена версия 0.1.7.10 или более новая, то сначала удалите драйвер (см. "Удаление драйвера фильтра"). Прямое обновление не поддерживается. 
- если у Вас установлена версия 0.1.7.9 или более старая, удаление не требуется. Новый инсталлер с интерфейсом GUI удалит эти версии автоматически. 
- затем установите новую версию так, как было описано ранее ("Установка драйвера фильтра").

Обновление драйвера устройства
- загрузите последнюю версию двоичного пакета драйвера устройства (libusb-win32-device-bin-x.x.x.x.tar.gz). 
- модифицируйте inf-файл, как описано в секции "Установка драйвера устройства". 
- откройте Device Manager и выберите устройство, которое собираетесь обновить.
- выберите 'Properties->Driver->Update'. Запретите автоматическую инсталляцию и выберите новый inf-файл вручную.
