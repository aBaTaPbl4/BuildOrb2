Это файл Readme для директории "tests" пакета AVR-USB, firmware-only
драйвера USB для микроконтроллеров AVR.

ЧТО НАХОДИТСЯ В ЭТОЙ ДИРЕКТОРИИ?
================================
Эта директория нужна только для разработки драйвера. Она содержит тесты для проверки,
компилируются ли как должно все ветки кода #ifdef, и увеличивается ли при этом размер
кода драйвера.
