	AVR-USB-MEGA16
	Эта отладочная плата позволяет разрабатывать низкоскоростные (low-speed) устройства USB на микроконтроллере ATMega16 компании Atmel. Идеально подходит для новичков в протоколе USB. Подробнее...
	

	Макетная плата AVR-USB-MEGA16 спроектирована для быстрой разработки низкоскоростных (low-speed) устройств USB на микроконтроллере ATMega16, при этом протокол USB реализован программно, без использования дополнительных специализированных чипов. Такое решение хорошо подходит для проектирования таких периферийных устройств для компьютера, которые не требуют высоких скоростей обмена по шине USB (подключение датчиков, устройств ввода, специализированных программаторов для чипов). Программная реализация протокола USB значительно упрощает принципиальную схему и снижает стоимость устройства в целом. Новичок в протоколе USB может легко спроектировать свое устройство USB и драйвера для него, поскольку имеется подробная документация и простые работоспособные примеры.
	
[Аппаратура]
- на плате имеются: макетное поле 8x11 с шагом между отверстиями 2.54 мм, разъем MINI USB, индикационный светодиод (используется в демонстрационных программах, поставляемых с платой), микроконтроллер ATMega16 в корпусе TQFP44, тактовая частота 16 МГц (используется кварц)
- размеры платы с макетным полем 64.8 x 30.7 мм, без макетного поля 45.4 x 30.7 мм (для уменьшения размеров макетное поле можно отрезать)
- программирование микроконтроллера возможно прямо на плате через стандартный (цоколевка Atmel) 6-контактный разъем для serial-программатора ISP
- для отладки программы микроконтроллера может используется 10-контактный JTAG-коннектор (цоколевка Atmel, может использоваться эмулятор JTAGICE mkII или аналогичные эмуляторы), также установленный на плате
- из 24 линий ввода/вывода ATMega16 занято 10 (2 - под сигналы USB, 1 для светодиода, 3 для программатора ISP, 4 для подключения JTAG-эмулятора), свободно для нужд пользователя 14
- из аппаратных ресурсов микроконтроллера для реализации протокола USB используется только прерывание INT0
- из ресурсов памяти программ для простых примеров используется 1464..2090 байт (зависит от типа устройства, опций оптимизации и версии компилятора), что составляет не более 13% памяти программ ATMega16
- питание платы осуществляется от 5 вольт шины USB

[Программное обеспечение]
- для ATMega16 используется пакет AVR-USB, разработанный компанией OBJECTIVE DEVELOPMENT Software GmbH, http://www.obdev.at/. Пакет распространяется по лицензии GPL, и доступнен для свободного скачивания на сайте компании здесь - http://www.obdev.at/products/avrusb/download.html. Переведенный на русский язык пакет AVR-USB доступен здесь - ...
- для компьютера (назовем его для краткости хост, как предлагает OBJECTIVE DEVELOPMENT) используется библиотека libusb-win32 для операционной системы Windows (2000, 2003, XP) и libusb для операционных систем семейства *nix (Mac OS, Linux и т. д.). Библиотека также поставляется по лицензии GPL. Скачать можно здесь - http://libusb.sourceforge.net/ (версия для *nix) и здесь - http://libusb-win32.sourceforge.net/ (для Windows)
- примеры программ для ATMega16 - USB-устройство пользовательского класса, класса HID, HID-мышь (для HID-устройств USB не нужны драйвера в операционных системах Windows), см. папку examples пакета AVR-USB. Программное обеспечение предназначено для компиляции с помощью gcc, есть также порт на систему IAR Embedded Workbench for Atmel AVR (заголовочный файл)
- примеры программ для хоста - драйверы и управляющие программы устройств USB, см. папку examples пакета AVR-USB. Программное обеспечение предназначено для компиляции с помощью gcc (и под Windows, и под *nix)
- плата поставляется с прошивкой, реализующей работу устройства HID (драйвер для такого USB-устройства в Windows 2000 и XP не требуется)
