Using the avrispmkII with avrdude on Windows

Posted on 2012-12-282012-12-28 by Elia

The open source AVR programming tool avrdude let’s you program your AVR microcontrollers from your Linux, Windows or Mac computer. It is easy to use on Linux or Mac but a little more effort is needed to get avrdude to talk to your avrispmkII on Windows.

We need to download and install the Windows port of the libusb driver so that avrdude can talk to the programmer (libusb is the usb driver that is used under Linux to talk to USB devices).

  1. First, download the latest version of libusb-win32 here: SourceForge libusb-win32 page
  2. Now extract the zip file to any location you like. Connect you avrispmkII, navigate to yourlocation\libusb-win32-bin-x.x.x.x\libusb-win32-bin-x.x.x.x\bin and run the inf-wizard.exe. Once it has opened, hit next.
  3. Select the avrispmkII entry as shown in the picture:

    Hit next.

  4. Make sure that you have selected the corret device and then hit next.

    Select a location to save the AVRISP_mkII.inf file (that’s the actual driver).
  5. Hit Install now…. Now the new driver is going to be installed.

    If Windows complains about the driver not being digitally signed, select Install anyway.

Now you can use the avrispmkII with avrdude on your Windows machine!

To use it with an ATtiny13, you could use:

D:\AVR\ATtiny13\RNG>avrdude -p t13 -c avrispmkII -P usb

avrdude: AVR device initialized and ready to accept instructions

Reading | ################################################## | 100% 0.03s

avrdude: Device signature = 0x1e9007

avrdude: safemode: Fuses OK

avrdude done. Thank you.

Note:

It is important that you not only specify the programmer with -c avrispmkII but that you also specify the USB-Port by using -P usb
